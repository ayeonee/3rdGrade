<고급 실습 과제 설명> - 20182682 김아연

기존의 삼각형으로만 이루어진 물체에서 벗어나 좀 더 복잡한 물체 데이터 파일을 고려하다가 마지막 수업시간에 튜토리얼 22번의 물체를 보았던 것이 생각나 이를 가져오기로 했다.
조명 모델 또한 튜토리얼 20번의 조명을 사용해 둘을 합쳐 보기로 했다. 튜토리얼 22번의 파일에 20번의 파일과 소스를 포함하였다.

먼저 ogldev 폴더 중 튜토리얼 20번에 있는 조명 계산을 위한 파일들을 튜토리얼 22번 폴더로 복사했다. 
lighting.fs, vs 파일, lighting_technique 헤더와 cpp 파일을 복사해 튜토리얼 22번 폴더로 붙여넣었다. 
비주얼 스튜디오를 실행한 뒤 튜토리얼 22번 프로젝트에 '기존 항목 추가'를 이용해 해당 파일들을 불러왔다.
(그 전에는 파일들을 복사하지 않고 바로 추가했었는데 각 파일들의 위치가 맞지 않아서 오류가 발생했었다.
또 아예 새 항목을 추가해 소스 내용들만 복사해도 튜토리얼 폴더에 파일이 생성되지 않아 역시 위치가 맞지 않았다.) 

튜토리얼 22번 cpp 파일에 lighting_technique.h를 추가했다. 그리고 이 헤더 안의 클래스(LightingTechnique)를 추가하려했는데 자꾸 이를 읽지 못하는 현상이 발생했다.
헤더 파일을 보니 전체가 비활성화 되는 문제가 발생했다. 여러 번의 시행 착오 끝에 튜토리얼 22번 cpp 파일 안에 이미 있던 lighting 코드 때문이란 것을 알고 이를 삭제해주었다.

Lighting_technique에서 불러오는 변수를 private 밑에 m_pEffect로 선언해주고 Init 함수 안에서 기존의 조명 계산을 하는 코드를 삭제했다.

RenderSceneCB 함수에 기존의 조명 계산 코드를 삭제하고 튜토리얼 20번 함수를 참고하여 새로운 코드를 추가하였다. 
PointLight을 만들었고 색을 조정했다. 기존의 튜토리얼 20번 함수보다 조명 변화를 더 빨리 하기 위해 scale 값을 수정하였다.
Pipeline을 이용해 조명 계산과 위치값, 물체의 크기, 움직이는 값을 지정하였다.

fs에서 ambient, diffuse, specular 조명을 다 합쳤고 Point Light값도 더해주고 최종적으로 계산이 끝난 TotalLight 값을 텍스쳐에 곱해주었다.
